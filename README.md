# Portofolio - Laila Dwi Kartika Sari

Personal portfolio website showcasing skills, experience, and certifications in UI/UX Design, Quality Assurance, and Data Science.

## Tech Stack

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Deployment

Deployed on [Vercel](https://vercel.com). Simply connect the GitHub repository and Vercel will auto-deploy.

## Project Structure

```
├── index.html          # Main HTML page
├── style.css           # All styles
├── script.js           # Interactive functionality
├── vercel.json         # Vercel deployment config
├── public/             # Static assets (images, certificates)
└── .gitignore          # Git ignore rules
```
